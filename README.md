# BATTLE OF DOOM — Neon Strike Launcher



## Controls

| Key | Action |
|-----|--------|
| WASD | Move |
| Shift | Sprint (uses Stamina) |
| LMB | Fire |
| RMB (hold) | Aim Down Sights |
| Space | Jump |
| R | Reload |
| E | Open Inventory |
| G | Hold Grenade |
| F | Interact (Shop/Doors) |
| N | Cycle Spectate Target (when Spectate is ON) |
